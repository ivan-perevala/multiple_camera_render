Support
=======

In case of any issues, please submit them alongside with a detailed description and steps to reproduce at the `GitHub Issues <https://github.com/ivan-perevala/multiple_camera_render/issues>`_.

Addon offers a built-in logging system that might help us to reproduce the issue.

.. image::
    https://raw.githubusercontent.com/ivan-perevala/multiple_camera_render/main/.github/images/submit-issue.png
   :alt: Submit Issue Button

It would open current log file in your default text editor, so you can copy it and paste into the issue description.

.. literalinclude:: log_example.inc

Don't be worried about the size of the log file, it is usually small and contains only relevant information.
For example, from this log file it is easy to understand which scene properties were used, which cameras were used, and what was the final result.
